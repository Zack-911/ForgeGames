v1.0.0
Creates a new game session. Returns a UUID that identifies this session — pass it to all other $game* functions.
$gameCreate[type;guildID;channelID;difficulty;timeoutSeconds;maxPlayers]
Arguments
typerequired
String
Game type: trivia | wordle | math | hangman | scramble | tictactoe | rps
guildIDrequired
Guild
Guild to associate with this session
channelIDrequired
Channel
Channel to associate with this session
difficulty
String
Difficulty: easy | medium | hard (default: medium)
timeoutSeconds
Number
Seconds before the session auto-expires (default: 30)
maxPlayers
Number
Max players allowed (default: 10)
Returns: String
